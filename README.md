# Random-Code
This repository is just a simple way for me to share code with others of random things.

## Contents

- [ANSI_Escape](./ANSI_Escape/) — Python scripts for ANSI escape codes and terminal coloring.
- [uname](./uname/) — C code and patches related to the `uname` command.
- [pn532-nfc](./pn532-nfc/) — Python script for bulk flashing NFC tags using the PN532 module on a Raspberry Pi.
